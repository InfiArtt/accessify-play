# Dialog Perangkat (Devices)

Dialog Perangkat (`D` di Command Layer) memungkinkan Anda mengalihkan pemutaran musik ke perangkat Spotify Connect lainnya (misalnya ponsel, smart speaker, atau komputer lain).

## Cara Menggunakan

1.  Buka dialog dengan menekan `D` di dalam Command Layer Accessify Play (`NVDA+G`).
2.  Daftar akan menampilkan semua perangkat yang terdeteksi. Perangkat yang sedang aktif akan ditandai dengan label "(Active)".
3.  Pilih perangkat tujuan dengan panah Atas/Bawah.
4.  Tekan `Enter` (atau klik tombol "Switch Device") untuk memindahkan pemutaran ke perangkat tersebut.

> **Catatan:** Accessify Play bertindak sebagai *controller*. Musik akan berhenti di perangkat ini dan mulai bermain di perangkat yang Anda pilih.
